### 简介

* promise是ES6的一门新的技术
* promise是js中进行异步编程的`新解决方法`
* 从语法上来说，promise是一个构造函数
* 从功能上来说，promise对象用来封装一个异步操作并且可以获取其成功或者失败的结果值

### 异步编程

* fs 文件操作
* 数据库操作
* AJAX
* 定时器

### 为什么使用promise
* 支持链式调用，解决回调地狱问题
* 指定回调函数的方式更加灵活
